#Uvod:
U ovom projektu, učenici će učiti o CSS pozicioniranju kroz gradnju robota.

#Online Izvori

 Predlažemo korištenje[trinket](https://trinket.io/) za stvaranje HTML i CSS online. Ovaj projekt sadrži slijedeće trinkete:

+ ['Build a Robot' starting point -- jumpto.cc/web-robot](http://jumpto.cc/web-robot)

Učenici mogu također iskroistiti ovo prazno polje trinket [(jumpto.cc/trinket-blank)](http://jumpto.cc/trinket-blank) da napišu svoj HTML & CSS, ili alternativno mogu koristiti ovaj predložak trinket [(jumpto.cc/trinket-template)](http://jumpto.cc/trinket-template).

Postoji također trinket koji sadrži uzorak za rješenje izazova

+ ['Build a Robot' Finished](https://trinket.io/html/00736c0e18)

#Offline izvori
Projek može biti[completed offline](https://www.codeclubprojects.org/en-GB/resources/webdev-working-offline/) po želji. Možete pristupiti resursima projekta klikom na vezu "Project materials" za ovaj projekt. Ova veza sadrži Odjeljak A 'projekt izvora ", što uključuje i sredstva koja će djeci trebati za dovršetak ovog projekta offline.Neka svaki učenik ima pristup kopiji ovog izvora. Ovaj odjeljak sadrži sljedeće datoteke:

+ Template/Template.html
+ Template/style.css
+ robot/Robot.html
+ Robot/style.css
+ Lots of .png images

Također možete pronaći ispunjenu verziju izazova ovog projekta je u odjeljku "Volonterski resursi ', koja sadrži:

+ RobotFinished.html
+ style.css
+ Lots of .png images

(Sve resurse gore su preuzeti kao projekt i volonter `.zip` files.)

#Ciljevi učenja
+ HTML id attribute;
+ Writing CSS:
	+ `#` id selector;
	+ position;
	+ top;
	+ left.

#Izazovi
+ "Napravi svoga robota" - primjenom CSS pozicioniranja napravi robota;
+ "Dodaj svoje slike" - dodavanjem više slike, s HTML id atributom povezane s CSS-om

#Dovrši projekt offline
Ako su djeca pri završetku ovog projekta offline, oni će morati spremiti slike koje namjeravaju koristiti u istoj mapi kao i njihov `Robot.html` datoteku.

Oni tada mogu samo dodati ime datoteke u njihovoj `<img>` oznaci:

```
<img id="hat" src="hat.png" />
```
